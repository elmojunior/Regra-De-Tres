# Calculadora de Regra de Três Simples

Uma calculadora web moderna, rápida e responsiva para resolver problemas de **Regra de Três Simples**. O projeto foi construído focando em performance e portabilidade, sendo inteiramente encapsulado em um único arquivo HTML, sem nenhuma dependência externa.

## 🚀 Funcionalidades

- **Zero Dependências:** Construído com puro HTML, CSS e JavaScript. Sem frameworks, bibliotecas externas ou assets para baixar.
- **Portabilidade Máxima:** Tudo (estrutura, estilos, scripts e ícones SVG) está contido num único arquivo (`regradetres.html`). Funciona completamente offline.
- **Modos Claro e Escuro:** Suporte a *Dark Mode* e *Light Mode*, com detecção automática baseada no tema do sistema do usuário e um botão interativo para alternância manual.
- **Design Responsivo:** Layout fluido e otimizado para dispositivos móveis, mantendo a melhor experiência de usuário sem a necessidade de rolar a página verticalmente no celular.
- **Formatação Numérica Brasileira:** Máscara inteligente que adiciona os pontos para separar os milhares e a vírgula para as casas decimais (ex: `1.234,56`), impedindo a digitação de caracteres inválidos.

## 💻 Como Usar

### Uso Online e Instalação (Recomendado)
Você pode acessar e instalar a calculadora diretamente no seu celular ou computador como um aplicativo nativo (PWA):
1. Acesse: [https://elmojunior.github.io/Regra-De-Tres/regradetres.html](https://elmojunior.github.io/Regra-De-Tres/regradetres.html)
2. Se estiver usando o Google Chrome (ou outro navegador compatível com PWA), aceite a sugestão ou vá no menu e clique em **Instalar Aplicativo**.
3. Uma vez instalado, o app aparecerá na sua bandeja de aplicativos e funcionará de maneira 100% offline.

### Uso Local e Offline
Se preferir, você também pode usar os arquivos da forma tradicional:
1. Baixe o arquivo `regradetres.html`.
2. Dê um duplo clique para abri-lo em qualquer navegador moderno (Chrome, Firefox, Safari, Edge).
3. Preencha os três valores conhecidos da sua regra de três nos campos indicados. O resultado do valor `X` será calculado instantaneamente de forma automática.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica.
- **CSS3:** Estilização, variáveis nativas (CSS Variables), gradientes, animações e Media Queries para responsividade.
- **JavaScript (Vanilla):** Lógica matemática de cálculo, manipulação de interface, máscara numérica e gerenciamento do tema com `localStorage`.
- **SVG:** Ícones renderizados inline garantindo a interface leve.
